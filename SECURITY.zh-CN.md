# 安全合规反馈

如果你在本仓库中发现安全漏洞、疑似侵权或其他合规问题，请发送邮件至 `octobus-opensource@chaitin.com` 向我们反馈。

请尽量提供以下信息：

- 仓库 URL 以及相关的分支、标签、提交记录或 Pull Request
- 受影响的文件、组件或功能
- 问题的详细描述
- 复现步骤、日志、截图或其他佐证材料
- 修复建议或紧急程度评估

我们通常会在收到新反馈后的 3 个工作日内回复，并尽快完成审查。

在问题调查和回复完成前，请勿公开披露未修复的漏洞或敏感的合规问题。

如果你不确定某问题是否应通过此渠道反馈，请先联系 `octobus-opensource@chaitin.com`，我们会协助转接。
